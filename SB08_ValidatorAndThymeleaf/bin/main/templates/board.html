글쓴이 : 나이 에러
컨텐츠 : 남 여 성별 에러

  1. th:action => 타임리프 템플릿 렌더링을 위해 사양되는 속성으로 값을 안 넣으면 같은 url에 post방식으로 요청하고 @{}안에 원하는 주소값을 넣는다
  2. th:object => 커맨드 객체를 지정해서 form에서 사용할 때 활용할 수 있다
  3. th:field => 선택 변수 식이라고 한다. th:object에서 선언한 객체 프로퍼티에 접근할 수 있다
  4. => *{name}는 ${PersonForm.name}를 의미
  5. th:if => 조건식에 사용한다.
  6. #뒤에는 객체가 붙는다. #fields는 th:field에서 지정한 속성명들을 모두 모아서 받는다.
